Ejercicio 1

¿Qué diferencia hay entre == y === en JavaScript? Mostrá un ejemplo donde den resultados distintos y explicá por qué.

== compara valor después de coerción. === compara valor y tipo sin coerción.

Ejemplo
console.log(2 == "2"); // true (coerción)
console.log(2 === "2"); // false (tipos diferentes)
💡 Conclusión: Usá === para evitar resultados inesperados.

Ejercicio 2

¿Qué es la coerción de tipos? Mostrá dos ejemplos de coerción implícita y uno de coerción explícita.

Es la conversión automática o manual de un valor a otro tipo de dato.

Coerción implícita (automática):

Ejemplos
console.log("5" - 3); // 2 (string a número)
console.log("2" + 2); // "22" (número a string)

Coerción explícita (manual):

Ejemplos
Number("123") // 123
String(456) // "456"
Boolean(0) // false

Ejercicio 3

¿Qué resultado devuelve una expresión como false || "hola" o "hola" && 0? Explicá el rol de los operadores lógicos en esos casos.

Ejemplos
false || "hola" // "hola" (primer truthy)
"hola" && 0 // 0 (primer falsy)

|| (OR): Devuelve el primer valor truthy. Si todos son falsy, devuelve el último.

&& (AND): Devuelve el primer valor falsy. Si todos son truthy, devuelve el último.

💡 Tip: No devuelven true/false, devuelven el valor original.

Ejercicio 4

¿Cuáles son las principales estructuras de control de flujo en JavaScript? Mencioná if, else, switch, y explicá brevemente en qué casos conviene usar cada uno.

if
if (edad >= 18) { console.log("Mayor de edad"); }
else
else { console.log("Menor de edad"); }
switch
switch(dia) {
  case "lunes": console.log("Inicio"); break;
  case "viernes": console.log("Fin"); break;
  default: console.log("Normal");
}
📌 Cuándo usar: if/else para condiciones o rangos. switch para comparar un mismo valor contra múltiples opciones exactas.

Ejercicio 5

¿Para qué sirve el operador ternario (condición ? a : b)? Reescribí un if simple con este operador.

if/else tradicional
if (edad >= 18) {
  console.log("Mayor");
} else {
  console.log("Menor");
}
Operador ternario

Ejercicio 6

¿Cómo funcionan los bucles for y while en JavaScript? Mostrá un ejemplo de cada uno.

for (se sabe cuántas veces)
for (let i = 0; i < 5; i++) {
  console.log("Vuelta " + i);
}
while (no se sabe cuántas veces)
let contador = 0;
while (contador < 5) {
  console.log("Contador: " + contador);
  contador++;
}
📌 Diferencia: for tiene contador incorporado. while solo tiene condición.

Ejercicio 7

¿Qué hacen break y continue dentro de un bucle? ¿Qué efecto tienen en la ejecución del código?

break (termina el bucle)
for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i); // 0,1,2,3,4
}
continue (salta a la siguiente vuelta)
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i); // 0,1,3,4
}
📌 Resumen: break sale del bucle. continue salta a la siguiente iteración.
