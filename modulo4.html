Ejercicio 1

¿Qué es un array en JavaScript y cómo se declara?

Un array es una estructura de datos que permite almacenar una colección ordenada de elementos. Los elementos pueden ser de cualquier tipo (números, strings, objetos, incluso otros arrays) y se accede a ellos mediante un índice numérico que comienza en 0.

Ejemplo
let frutas = ["manzana", "banana", "naranja"];
let numeros = [1, 2, 3, 4, 5];
let mixto = [42, "texto", true, { clave: "valor" }];
let vacio = []; // array vacío
// También se puede usar el constructor Array()
let numeros2 = new Array(5); // crea un array con 5 posiciones vacías

Ejercicio 2

¿Cómo accedés al tercer elemento de un array?

Se accede mediante el índice numérico. Como los índices comienzan en 0, el tercer elemento está en la posición 2.

Ejemplo
let colores = ["rojo", "azul", "verde", "amarillo"];
console.log(colores[2]); // "verde"

Ejercicio 3

¿Qué métodos de array son los más comunes en JavaScript? Explicá para qué sirve cada uno y cómo podrías usarlos para recorrer, agregar o quitar elementos.

Agregar elementos:

  • push() - agrega uno o más elementos al final del array
  • unshift() - agrega uno o más elementos al inicio del array

Quitar elementos:

  • pop() - elimina el último elemento y lo devuelve
  • shift() - elimina el primer elemento y lo devuelve
  • splice() - elimina, reemplaza o inserta elementos en cualquier posición

Recorrer / transformar:

  • forEach() - ejecuta una función por cada elemento
  • map() - crea un nuevo array transformando cada elemento
  • filter() - crea un nuevo array con los elementos que cumplen una condición
  • reduce() - reduce el array a un solo valor (suma, promedio, etc.)

Buscar:

  • find() - devuelve el primer elemento que cumple una condición
  • some() - verifica si al menos un elemento cumple una condición
  • every() - verifica si todos los elementos cumplen una condición
  • includes() - verifica si un valor existe en el array
Ejemplos
let numeros = [1, 2, 3];
numeros.push(4); // [1, 2, 3, 4]
numeros.pop(); // [1, 2, 3]
numeros.forEach(n => console.log(n)); // 1, 2, 3
let dobles = numeros.map(n => n * 2); // [2, 4, 6]
let pares = numeros.filter(n => n % 2 === 0); // [2]

Ejercicio 4

¿Qué método usarías para eliminar el último elemento de un array? ¿Y el primero?

pop() elimina el último elemento. shift() elimina el primer elemento.

Ejemplo
let frutas = ["manzana", "banana", "naranja"];
frutas.pop(); // ["manzana", "banana"]
frutas.shift(); // ["banana"]

Ejercicio 5

¿Qué pasa si accedés a una posición inexistente en un array?

Devuelve undefined. No genera error, simplemente la posición no tiene valor asignado.

Ejemplo
let frutas = ["manzana", "banana"];
console.log(frutas[5]); // undefined

Ejercicio 6

¿Cómo verificás si un valor está presente en un array o en un Set?

Para arrays se usa includes() o indexOf(). Para Sets se usa has().

Ejemplo
let numeros = [1, 2, 3, 4];
numeros.includes(3); // true
numeros.includes(5); // false
let setNumeros = new Set([1, 2, 3, 4]);
setNumeros.has(3); // true
setNumeros.has(5); // false

Ejercicio 7

¿Qué es un Set? ¿En qué casos conviene usarlo en lugar de un array?

Un Set es una colección de valores únicos. No permite duplicados y no tiene índices numéricos. Conviene usarlo cuando:

  • Se necesita garantizar que no haya valores repetidos
  • Se realizan muchas búsquedas de existencia (método has() es más rápido que includes())
  • No importa el orden ni la posición de los elementos
Ejemplo
let idsUnicos = new Set([1, 2, 2, 3, 3, 3]);
console.log(idsUnicos); // Set(3) {1, 2, 3}
// Eliminar duplicados de un array
let conDuplicados = [1, 2, 2, 3, 3];
let sinDuplicados = [...new Set(conDuplicados)]; // [1, 2, 3]

Ejercicio 8

¿Qué es un Map y qué lo diferencia de un objeto?

Un Map es una colección de pares clave-valor donde las claves pueden ser de cualquier tipo (objetos, funciones, números, etc.), a diferencia de los objetos comunes que solo permiten claves de tipo string o Symbol.

Diferencias principales
let mapa = new Map();
mapa.set("nombre", "Juan");
mapa.set(123, "valor numérico como clave");
mapa.set({ id: 1 }, "objeto como clave");
// Map mantiene el orden de inserción
// Map tiene métodos .size, .has(), .delete(), .clear()
// Objeto común: solo strings o Symbols como claves

Ejercicio 9

¿Cómo recorrerías un array con un bucle? Mostrá al menos dos formas diferentes.

Forma 1: for tradicional
let frutas = ["manzana", "banana", "naranja"];
for (let i = 0; i < frutas.length; i++) {
console.log(frutas[i]);
}
Forma 2: forEach
frutas.forEach(function(fruta) {
console.log(fruta);
});
Forma 3: for...of
for (let fruta of frutas) {
console.log(fruta);
}

Ejercicio 10

¿Qué es un objeto en JavaScript y cómo se declara?

Un objeto es una colección de propiedades, donde cada propiedad tiene una clave (string) y un valor. Se puede declarar usando llaves {}.

Ejemplo
let persona = {
nombre: "Juan",
edad: 30,
profesion: "programador",
saludar: function() {
console.log("Hola");
}
};
let vacio = {}; // objeto vacío

Ejercicio 11

¿Cuál es la diferencia entre acceder a una propiedad con . y con []?

La notación con punto (.) se usa cuando el nombre de la propiedad es conocido y válido como identificador (sin espacios, no comienza con número). La notación con corchetes ([]) permite usar strings con espacios, variables o expresiones.

Ejemplo
let persona = { nombre: "Juan", "apellido materno": "Perez" };
console.log(persona.nombre); // "Juan"
console.log(persona["apellido materno"]); // "Perez"
let clave = "nombre";
console.log(persona[clave]); // "Juan" (usando variable)

Ejercicio 15

¿Cómo se recorren las propiedades de un objeto? ¿Para qué sirve for...in?

Se puede usar for...in para recorrer las propiedades enumerables de un objeto.

Ejemplo
let persona = { nombre: "Juan", edad: 30, ciudad: "Buenos Aires" };
for (let clave in persona) {
console.log(clave + ": " + persona[clave]);
}
// Resultado:
// nombre: Juan
// edad: 30
// ciudad: Buenos Aires

for...in sirve para iterar sobre las propiedades de un objeto, útil cuando no se conocen los nombres de las propiedades de antemano.

Ejercicio 16

¿Qué devuelven Object.keys(), Object.values() y Object.entries()?

Ejemplo
let persona = { nombre: "Juan", edad: 30, ciudad: "Buenos Aires" };
Object.keys(persona); // ["nombre", "edad", "ciudad"]
Object.values(persona); // ["Juan", 30, "Buenos Aires"]
Object.entries(persona); // [["nombre","Juan"], ["edad",30], ["ciudad","Buenos Aires"]]
  • Object.keys() - devuelve un array con las claves (nombres de las propiedades)
  • Object.values() - devuelve un array con los valores
  • Object.entries() - devuelve un array de arrays, cada uno con [clave, valor]

Ejercicio 17

¿Qué diferencia hay entre un array y un objeto en cuanto a estructura y uso?

  • Array: Colección ordenada de elementos accesibles por índice numérico (0, 1, 2...). Se usa cuando el orden importa y se trabaja con listas de elementos del mismo tipo.
  • Objeto: Colección de propiedades con claves de tipo string. Se usa cuando se necesita describir una entidad con atributos nombrados.
Ejemplo
// Array: lista de elementos
let colores = ["rojo", "azul", "verde"];
colores[0]; // "rojo" (por posición)
// Objeto: entidad con atributos
let persona = { nombre: "Juan", edad: 30 };
persona.nombre; // "Juan" (por clave)

Ejercicio 18

¿Qué es la desestructuración de objetos y para qué sirve?

Es una sintaxis que permite extraer propiedades de un objeto y asignarlas a variables de forma más concisa.

Ejemplo
let persona = { nombre: "Juan", edad: 30, ciudad: "Buenos Aires" };
// Sin desestructuración
let nombre = persona.nombre;
let edad = persona.edad;
// Con desestructuración
let { nombre, edad } = persona;
console.log(nombre); // "Juan"
console.log(edad); // 30
// Con alias
let { nombre: nombrePersona, edad: edadPersona } = persona;
// Valor por defecto
let { telefono = "no disponible" } = persona;

Sirve para escribir código más limpio y extraer múltiples propiedades en una sola línea.

Ejercicio 19

¿Cómo se usan los operadores spread (...) y rest (...) con objetos?

Spread (...) permite expandir las propiedades de un objeto dentro de otro. Se usa para copiar o combinar objetos.

Rest (...) permite agrupar las propiedades restantes en un nuevo objeto.

Spread (...)
let persona = { nombre: "Juan", edad: 30 };
let direccion = { ciudad: "Buenos Aires", pais: "Argentina" };
let completo = { ...persona, ...direccion };
// { nombre: "Juan", edad: 30, ciudad: "Buenos Aires", pais: "Argentina" }
let copia = { ...persona }; // copia superficial
Rest (...)
let { nombre, ...resto } = { nombre: "Juan", edad: 30, ciudad: "Buenos Aires" };
console.log(nombre); // "Juan"
console.log(resto); // { edad: 30, ciudad: "Buenos Aires" }
