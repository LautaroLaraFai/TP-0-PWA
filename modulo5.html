Ejercicio 1

¿Qué es la programación funcional y cómo se relaciona con los métodos de array en JavaScript?

La programación funcional es un paradigma de programación declarativo basado en el uso de funciones matemáticas para transformar datos. En JavaScript, la programación funcional prioriza la inmutabilidad y el uso de funciones de orden superior. Los métodos de array como map(), filter() y reduce() demuestran este enfoque porque, en lugar de modificar el array original, devuelven un nuevo array con los cambios aplicados, manteniendo los datos originales intactos.

Ejercicio 2

¿Cuál es la diferencia entre .map(), .filter() y .reduce()?

  • .map(): Aplica una función a cada elemento del array y devuelve un nuevo array con los resultados.
  • .filter(): Crea un nuevo array con los elementos que cumplen una condición.
  • .reduce(): Reduce el array a un único valor (suma, promedio, concatenación, etc.).
Ejemplo
let numeros = [1, 2, 3, 4, 5];
let dobles = numeros.map(n => n * 2); // [2, 4, 6, 8, 10]
let pares = numeros.filter(n => n % 2 === 0); // [2, 4]
let suma = numeros.reduce((acc, n) => acc + n, 0); // 15

Ejercicio 3

¿Qué es la inmutabilidad y por qué es importante en la programación funcional?

La inmutabilidad es la propiedad de los datos que impide que sean modificados después de su creación. En lugar de cambiar los datos existentes, la programación funcional promueve crear nuevos datos a partir de los originales. Esto evita efectos secundarios no deseados, hace el código más predecible y facilita la depuración.

Ejemplo
let numeros = [1, 2, 3];
let nuevos = numeros.map(n => n * 2); // Crea un nuevo array
console.log(numeros); // [1, 2, 3] (original intacto)
console.log(nuevos); // [2, 4, 6] (nuevo array)

Ejercicio 4

¿Cómo funcionan los callbacks en JavaScript? Explicá con un ejemplo.

Un callback es una función que se pasa como argumento a otra función y se ejecuta cuando esa función termina. Es una forma de controlar el flujo de ejecución, especialmente en operaciones asincrónicas.

Ejemplo
function saludar(nombre) {
console.log("Hola " + nombre);
}
function procesarEntradaUsuario(callback) {
let nombre = prompt("Por favor ingresa tu nombre:");
callback(nombre);
}
procesarEntradaUsuario(saludar); // "Hola [nombre]"

En este ejemplo, saludar es el callback. Se pasa a procesarEntradaUsuario y se ejecuta después de obtener el nombre.

Ejercicio 5

¿Qué es el método .some() y en qué casos se utiliza?

.some() es un método que recorre un array y devuelve true si al menos un elemento cumple con la condición especificada. Si ningún elemento cumple, devuelve false. Se utiliza para verificar si existe al menos un elemento que cumpla una condición.

Ejemplo
let numeros = [1, 2, 3, 4, 5];
let hayMayorQue3 = numeros.some(n => n > 3); // true
let hayNegativo = numeros.some(n => n < 0); // false

Ejercicio 6

¿Qué es el método .find() y cómo se diferencia de .filter()?

.find() devuelve el primer elemento que cumple la condición. .filter() devuelve todos los elementos que cumplen la condición en un nuevo array. Si no encuentra nada, .find() devuelve undefined y .filter() devuelve un array vacío.

Ejemplo
let numeros = [1, 2, 3, 4, 5];
let primerMayor = numeros.find(n => n > 2); // 3
let todosMayores = numeros.filter(n => n > 2); // [3, 4, 5]

Ejercicio 7

¿Cómo se puede combinar .map() con .filter() para transformar y filtrar datos?

Se pueden encadenar ambos métodos para primero filtrar los elementos que interesan y luego transformarlos.

Ejemplo
const numeros = [1, 2, 3, 4, 5, 6];
const resultado = numeros
.filter(num => num % 2 !== 0) // Filtra impares: [1, 3, 5]
.map(num => num * 2); // Multiplica por 2: [2, 6, 10]
console.log(resultado); // [2, 6, 10]

Ejercicio 8

¿Qué es la desestructuración de arrays y para qué sirve?

La desestructuración de arrays es una sintaxis que permite extraer valores de un array y asignarlos a variables de forma más concisa. Sirve para acceder rápidamente a elementos específicos sin usar índices repetidamente.

Ejemplo
let colores = ["rojo", "azul", "verde"];
// Sin desestructuración
let primero = colores[0];
let segundo = colores[1];
// Con desestructuración
let [rojo, azul, verde] = colores;
console.log(rojo); // "rojo"
console.log(azul); // "azul"
// Omitir elementos
let [primero, , tercero] = colores;
console.log(primero); // "rojo"
console.log(tercero); // "verde"

Ejercicio 9

¿Cómo se puede usar el operador spread (...) con arrays?

El operador spread (...) permite expandir los elementos de un array en lugares donde se esperan múltiples elementos. Se usa para combinar arrays, copiar arrays o pasar elementos como argumentos.

Ejemplo
let miArray1 = [1, 2, 3];
let miArray2 = [4, 5, 6];
// Combinar arrays
let combinado = [...miArray1, ...miArray2]; // [1, 2, 3, 4, 5, 6]
// Copiar array
let copia = [...miArray1]; // [1, 2, 3]
// Pasar elementos como argumentos
let maximo = Math.max(...miArray1); // 3

Ejercicio 10

¿Qué es el método .flatMap() y en qué casos es útil?

.flatMap() primero aplica una función a cada elemento (como .map()) y luego aplana el resultado en un solo nivel (como .flat()). Es útil cuando cada elemento del array se transforma en múltiples elementos y se quiere obtener un array plano como resultado.

Ejemplo
let frases = ["hola mundo", "javascript es genial"];
// Con .map() + .flat()
let palabras = frases.map(frase => frase.split(" ")).flat();
// ["hola", "mundo", "javascript", "es", "genial"]
// Con .flatMap() (mismo resultado)
let palabras2 = frases.flatMap(frase => frase.split(" "));
// También sirve para filtrar y mapear en un solo paso
let numeros = [1, 2, 3];
let duplicados = numeros.flatMap(n => [n, n]); // [1, 1, 2, 2, 3, 3]

Casos de uso: Transformar arrays de arrays en un solo nivel, separar strings en palabras, duplicar elementos, o combinar mapeo y filtrado en una sola operación.
